# 버튼을 잡아라!(C# 코딩)
## 개요
-이 작품은 C# WinForms를 이용하여 만든 버튼을 잡아라!게임 마우스가 버튼에 다가가면 도망가도록 만든 프로젝트이다.
## 핵심기능
- 버튼 텍스트를 나를 잡아봐로 설정
- 버튼이 화면밖으로 나가지 않도록 범위를 제한
- 마우스가 버튼위로 가면 다른 위치로 이동하도록 설정
## 화면 구성
-form 1개
-Button 1개
-버튼 이름을 Target 텍스트는 나를 잡아봐
